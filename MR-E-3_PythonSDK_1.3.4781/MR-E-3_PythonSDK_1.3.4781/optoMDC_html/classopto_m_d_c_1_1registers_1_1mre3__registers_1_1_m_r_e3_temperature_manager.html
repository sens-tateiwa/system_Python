Detailed Description

    Device Functionality - Device Temperature Readout
    System ID: 0x22
    If this threshold is reached the output of the driver is disabled to cool down.
    Temperature has to fall by 10 degrees so that the output is enabled again
    +---------------------+------+-------------+------+-------+---------+------------------------------------------+
    | Register Name                   | Id   | Type        | Unit | Range | Default | Comment                      |
    +=====================+======+=============+======+=======+=========+==========================================+
    |actuator_temperature             | 0x00 |uint 8-bit  |°C    |       |N/A       |Degrees Celsius               |
    +---------------------+------+-------------+------+-------+---------+------------------------------------------+
    |actuator_temp_threshold          | 0x01 |uint 8-bit  |°C    |       |85        |Degrees Celsius               |
    +---------------------+------+-------------+------+-------+---------+------------------------------------------+
    |output_stage_board_temp          | 0x02 |uint 8-bit  |°C    |       |N/A       |Degrees Celsius               |
    +---------------------+------+-------------+------+-------+---------+------------------------------------------+
    |output_stage_board_temp_threshold| 0x03 |uint 8-bit  |°C    |       |85        |Degrees Celsius               |
