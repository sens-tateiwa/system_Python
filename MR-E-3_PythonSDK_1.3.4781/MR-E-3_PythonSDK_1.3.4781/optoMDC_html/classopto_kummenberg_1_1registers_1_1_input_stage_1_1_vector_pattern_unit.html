| Register Name  | Id   | Type        | Unit               | Range       | Default | Comment                |

|unit            | 0x00 | uint 32-bit | None               | 0 to 2      | 0       | See registers table    |
+----------------+------+-------------+--------------------+-------------+---------+------------------------+
|run             | 0x01 | uint 32-bit | bool               | True / False| 0       |                        |
+----------------+------+-------------+--------------------+-------------+---------+------------------------+
|start           | 0x02 | float 32-bit|                    |             |         |                        |
+----------------+------+-------------+--------------------+-------------+---------+------------------------+
|end             | 0x03 | float 32-bit|                    |             |         |                        |
