Detailed Description

* Device Functionality - Resonant Axis Amplitude Control
* System ID: 0x35
*
* ### Register Map:
*
*  Address | Name                             | Default          | Description                                                                   | Type        | Access
*  --------|----------------------------------|------------------|-------------------------------------------------------------------------------|-------------|------------------
*  0x3500  | Enable control                   | false or 0       | Enable control of resonant axis                                               | bool        | read write
*  0x3501  | Force res. frequency measurement | N/A              | Reading triggers resonant frequency measurement  <br/>(Important for command timeout - procedure takes cca 300ms) | float       | read only
*  0x3502  | Resonant (natural) frequency     | N/A              | Returns resonant frequency of the resonant axis based on startup measurement  | float       | read only
*  0x3503  | Actual (driven) frequency        | N/A              | Actual driving frequency in Hz <br/>(not recommended to change when control is enabled)             | float       | read write
*  0x3504  | Actual mechanical amplitude      | N/A              | Actual driving amplitude (angle) in degrees                                   | float       | read only
*  0x3505  | Desired mechanical amplitude     | 2.5              | Desired driving amplitude (angle) in degrees                                  | float       | read write
*  0x3506  | Actual field of view             | N/A              | Actual field of view (angle) in degrees                                       | float       | read only
*  0x3507  | Desired field of view            | 10.0             | Desired field of view (angle) in degrees                                      | float       | read write
*  0x3508  | Current driving resonant axe     | 0.07 or 0.12 **  | Driving current setting in A <br/>(not recommended to change default value; limited to 0.3A)        | float       | read write
*  0x3509  | Frequency shift                  | 2.5              | Frequency shift for initial control loop (Hz)                                 | float       | read write
*  0x350A  | Frequency delta coefficient      | 0.3              | Coefficient adjusting step in calculation of control frequency                | float       | read write
*  0x350B  | Frequency tolerance              | 20               | Tolerance in % for limiting frequency control range                           | float       | read write
*  0x350C  | Resonance slope                  | -0.5             | Slope of amplitude frequency characteristic (deg/Hz)                          | float       | read write
*  0x350D  | Time window                      | 0.005            | Integration time for amplitude measurement in seconds                         | float       | read write
*  0x350E  | Averaging cycles                 | 10               | Number of cycles for average measurement of amplitude (at frequency 10kHz)    | float       | read write
*  0x350F  | Control cycles                   | 20               | Number of wait cycles before frequency control applies changes  <br/>(real timing = Time window * Averaging cycles * Control cycles)                             | float       | read write
*
*  Note 1 : Force res. frequency measurement takes cca 200ms, adjust communication timeouts accordingly
*  Note 2 : Default driving current is set depending on type of resonant mirror
*/
